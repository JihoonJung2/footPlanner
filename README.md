# ⚽ FootPlanner - 축구 일정 사이트

간단하고 직관적인 **축구 경기 일정 & 즐겨찾기 관리 웹앱**입니다.  
팀별 일정 확인과 즐겨찾기 기능을 통해 원하는 경기 정보를 빠르게 확인할 수 있습니다.

---

## 🛠️ 기술 스택
- **Frontend:** React
- **Backend:** Node.js
- **Database:** MySQL 
- **인증:** JWT 토큰 기반 로그인
- **환경변수 관리:** `.env` 파일 사용
- **외부 API:** 축구 경기 일정 데이터 외부 API 연동

---

## 🚀 주요 기능
- 4대 리그 모든 대회 일정 확인
- 즐겨찾기 팀 추가/삭제
- JWT 기반 로그인/회원 관리
- 외부 API를 통한 경기 데이터 실시간 불러오기
- 순위 확인
